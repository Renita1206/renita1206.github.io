Arcade Games

This project is a snall collection of classic arcade games. The repository currently includes two games: Snake and Ping Pong. These games are implemented using HTML for the structure and JavaScript for the game logic.

The Snake game allows the player to control a snake and navigate it around the game board to eat food and grow longer. The game keeps track of the player's score as they consume food and provides a challenging experience as the snake's length increases.

Ping Pong is a two-player game where each player controls a paddle to hit a ball back and forth across the screen. The objective is to prevent the ball from passing the player's paddle while trying to make the opponent miss. The game keeps track of the score for each player and offers an engaging multiplayer experience.

This project was independently built with the primary aim of learning javascript and getting an introduction to web development. This also allowed me to explore the basics of game development.
